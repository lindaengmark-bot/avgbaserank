# SEO Keyword Rank Tracker v3

Den här versionen återställer de mer kompletta sammanställningarna och stöder samtidigt två filformat:

- Wide format, en kolumn per månad
- Long format, med Start Date, Keyword och Google Base Rank eller Google

## Sammanställningar

- Tracked keywords
- Ranked keywords
- No-rank keywords
- Visibility %
- Average rank, ranked only
- Average rank, incl. no-rank
- Top 3
- Top 10
- Top 20
- Top 100
- Månatlig förändring mot föregående månad
- Vinnare och tappare mellan två valfria månader

## Kör lokalt

```bash
pip install -r requirements.txt
streamlit run app.py
```
